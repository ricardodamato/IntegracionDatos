# Actividad - Integración Multidimensional de Datos

## 📌 Introducción
En esta actividad se desarrolla un proceso de integración multidimensional de datos en una entidad del sector Fintech, con el fin de optimizar la asignación de clientes a sucursales y mejorar la toma de decisiones.

## 🎯 Objetivo
Reducir el número de sucursales mediante la agrupación de solicitantes de crédito en cinco clusters, utilizando técnicas de análisis multidimensional.

## 📊 Descripción
Se realiza la integración de datos considerando múltiples variables numéricas como edad, ingresos, egresos y características del crédito. A partir de estas variables, se agrupan los solicitantes en cinco sucursales (clusters), permitiendo identificar patrones de comportamiento.

Adicionalmente, se analiza:
- Número de clientes por sucursal  
- Municipios de procedencia  
- Porcentaje de preaprobación y prenegación  

## 🛠️ Metodología
- Algoritmo de clustering **K-Medoids** con 5 semillas iniciales  
- Variables utilizadas:
  - Edad  
  - Hijos  
  - Personas a cargo  
  - Estrato  
  - Ingresos  
  - Egresos  
  - Monto  
  - Plazo  
  - Cuota  
- Visualización de resultados mediante gráficos multidimensionales (ej. gráficos de araña)  
- Publicación de resultados en GitHub para apoyo en la toma de decisiones
