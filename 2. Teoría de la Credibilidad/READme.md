# Actividad - Integración de Datos y Teoría de la Credibilidad

## 📌 Introducción
En esta actividad se aborda el proceso de integración de datos en una FINTECH, utilizando la teoría de la credibilidad para evaluar la relación entre diferentes bases de datos y mejorar la estimación del riesgo.

## 🎯 Objetivo
Integrar bases de datos internas y externas, evaluando su credibilidad, para obtener una mejor caracterización de la distribución agregada de pérdidas.

## 📊 Descripción
Se realiza la caracterización de la distribución agregada de pérdidas utilizando métricas de tendencia central y dispersión. Posteriormente, se estima la credibilidad de la base de datos externa respecto a la interna, y se integran ambas para analizar el comportamiento conjunto de las pérdidas.

## 🛠️ Metodología
- Método de Montecarlo para el enriquecimiento de datos  
- Aplicación de la teoría de la credibilidad  
- Análisis estadístico mediante:
  - Media  
  - Varianza  
  - Medidas de dispersión y tendencia central
