# 📊 Modelamiento y Gestión de Riesgo Operacional - FINTECH

## 📌 Descripción
Este repositorio contiene el desarrollo de diferentes actividades relacionadas con el modelamiento del riesgo operacional en una entidad FINTECH, abordando técnicas estadísticas, simulación y métodos de integración de datos.

## 🎯 Objetivo General
Analizar, modelar y gestionar el riesgo operacional mediante herramientas cuantitativas, con el fin de apoyar la toma de decisiones en contextos financieros reales.

## 📚 Contenido del Repositorio

### 1. Caracterización de Variables Aleatorias
Análisis de distribuciones, medidas estadísticas y simulación de variables de frecuencia y severidad.

### 2. Integración de Datos - Teoría de la Credibilidad
Evaluación e integración de bases de datos utilizando métricas estadísticas y teoría de la credibilidad.

### 3. Integración en Entidad de Salud
Reubicación de pacientes mediante modelos de afinidad, credibilidad y métodos de aceptación/rechazo.

### 4. Integración Multidimensional
Segmentación de clientes utilizando clustering (K-Medoids) y análisis de variables financieras.

### 5. Evaluación de Riesgo Operacional
Construcción de matrices de riesgo, estimación de pérdidas esperadas y cálculo de OpVaR.

### 6. Gestión Prospectiva de Riesgos
Aplicación de modelos de clustering (K-Means) sobre distribuciones de pérdidas, incorporando indicadores de valor capturado.

## 🛠️ Herramientas utilizadas
- Python (Google Colab)
- Excel
- Métodos estadísticos y probabilísticos
- Simulación Montecarlo
- Clustering (K-Means y K-Medoids)
