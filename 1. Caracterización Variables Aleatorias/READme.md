# Actividad - Modelamiento del Riesgo Operacional en una FINTECH

## 📌 Introducción
En esta actividad se analiza el riesgo operacional asociado a fallos tecnológicos en una FINTECH, con el fin de estimar las pérdidas derivadas de errores en las transacciones realizadas por los clientes a través de canales electrónicos.

## 🎯 Objetivo
Estimar las pérdidas asociadas al riesgo tecnológico mediante el análisis de variables de frecuencia y severidad, utilizando simulación Montecarlo.

## 📊 Descripción
Se realiza la caracterización de la distribución agregada de pérdidas (LDA), considerando métricas clave como pérdidas esperadas, no esperadas y catastróficas. Además, se analizan medidas estadísticas como media, varianza, asimetría y curtosis para entender el comportamiento de los datos.

## 🛠️ Metodología
- Simulación mediante el método de Montecarlo  
- Análisis de variables aleatorias de frecuencia y severidad  
- Evaluación con métricas estadísticas:
  - Media  
  - Varianza  
  - Asimetría  
  - Curtosis
