# Actividad - Gestión Prospectiva de Fallas Tecnológicas (Reto Parcial II)

## 📌 Introducción
En esta actividad se desarrolla un modelo de gestión prospectiva del riesgo en una entidad FINTECH, incorporando un enfoque innovador basado en el potencial de captura de Gases de Efecto Invernadero (GEI) a partir del análisis de fallas tecnológicas.

## 🎯 Objetivo
Implementar un modelo que permita anticipar riesgos operacionales y estimar el valor capturado financiero, relacionándolo con indicadores de captura de GEI.

## 📊 Descripción
Se utiliza una base de datos de eventos de riesgo asociados a fallas tecnológicas para modelar la distribución agregada de pérdidas (LDA). A partir de esta distribución, se aplica un modelo de clustering para identificar patrones de riesgo y estimar el valor capturado financiero derivado de una gestión prospectiva.

Este valor se interpreta como un indicador asociado a la captura de gases de efecto invernadero (GEI), integrando así una dimensión ambiental en la gestión del riesgo.

## 🛠️ Metodología
- Modelamiento de la distribución agregada de pérdidas (LDA)  
- Aplicación del algoritmo K-Means para segmentación de riesgos  
- Implementación en Google Colab  
- Estimación del valor capturado financiero como indicador prospectivo
