# Actividad - Integración de Datos en Entidad Prestadora de Salud

## 📌 Introducción
En esta actividad se analiza el proceso de integración de datos en una entidad prestadora de salud, con el objetivo de optimizar la prestación de servicios mediante la reubicación de pacientes ante el cierre de una sucursal.

## 🎯 Objetivo
Determinar la sucursal con mayor afinidad a la sede de Sabaneta y reubicar a los pacientes utilizando métodos de integración de datos y teoría de la credibilidad.

## 📊 Descripción
Se evalúa la afinidad entre la sucursal de Sabaneta y otras sucursales mediante la estimación de la credibilidad, tomando como referencia una variable asociada a pacientes con diabetes. Posteriormente, se realiza la reubicación de pacientes utilizando el método de valor de pertenencia y se compara con el método de aceptación y rechazo.

Además, se analizan las medidas de tendencia central y dispersión antes y después del proceso de integración para evaluar el impacto en la distribución de los datos.

## 🛠️ Metodología
- Teoría de la credibilidad para estimar afinidad entre sucursales  
- Métodos de integración:
  - Valor de pertenencia  
  - Aceptación y rechazo  
- Análisis estadístico mediante:
  - Media  
  - Medidas de dispersión
